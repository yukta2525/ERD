# Final Queries

This folder contains your polished, final queries for project submission.

## Before Moving Queries Here

✅ Query has been tested and produces correct results
✅ Query is well-commented and formatted
✅ Results have been validated by the team
✅ Query follows SQL best practices
✅ Purpose and findings are documented

## Final Submission Checklist

- [ ] All final queries are in this folder
- [ ] Each query file includes purpose, author, and results sections
- [ ] Query results are documented (in comments or in `/results` folder)
- [ ] Main README is updated with key findings
- [ ] Team notes document the collaboration process
- [ ] All team members have reviewed the queries
- [ ] Code is properly formatted (use Shift+Alt+F in VS Code)

## Naming Convention

Use descriptive names that indicate the business question:
- ✅ `top_rated_restaurants_by_city.sql`
- ✅ `user_engagement_analysis.sql`
- ✅ `seasonal_review_trends.sql`
- ❌ `final_query_1.sql`
- ❌ `project.sql`
